IFNULL and COALESCE

https://codewithmosh.com/courses/525068/lectures/9590263

USE database SELECT col1, IFNULL(col2, 'Custom Message') AS something -- COALESCE(col2, col3, col4, 'Custom Message') AS something FROM table

Note IFNULL replace null with one message COALESCE will 1st check col2 for null, then col3 for null ... and if all null, then give the custom message In case, between various col2, col3 we find value, we fill that in the something col.
